**Purpose**: This file outlines the process for integrating Auth0 authentication into an existing application.

**What it does**:
- Integrates Auth0 services to manage application user authentication.
- Requires the user to provide the necessary Auth0 domain and client ID to complete the setup.

**Gotchas**:
- The setup is entirely dependent on the user having pre-configured credentials (domain and client id) within an external Auth0 account.
