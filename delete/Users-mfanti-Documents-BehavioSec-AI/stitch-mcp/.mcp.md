**Purpose**: The file defines the configuration and connection details for the Stitch Multi-Cloud Platform (MCP) server.

**What it does**:
- Configures an HTTP connection to the specified Google Cloud endpoint (`https://stitch.googleapis.com/mcp`).
- Provides the necessary authentication mechanism by setting the `X-Goog-Api-Key` header.

**Key exports**:
- stitch: The primary configuration block detailing the type, URL, and required headers for connecting to the Stitch MCP service.

**Gotchas**:
- The API key is hardcoded within this file, which is a security risk; in production environments, credentials should be loaded from environment variables or a secure secret manager rather than stored directly in configuration files.
